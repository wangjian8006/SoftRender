<h1>一个C++实现的软渲染例子</h1>
<h2>介绍</h2>
模拟Unity计算，封装了旋转位移缩放等接口变换，这样所有的计算可以用unity来验证是否正确。</br>
实现了欧拉角、四元数、旋转矩阵的互相转换</br>
MVP矩阵的计算</br>
深度缓存</br>
简单的CVV裁剪</br>
线框模式</br>
简单的扫描线画三角形</br>
简单的材质系统，修改材质属性，并且可以替换不同的shader</br>
简单的Shader系统，可以实现顶点着色器与片元着色器</br>
两个简单shader，一个是顶点颜色，一个是贴图颜色</br>
一个BlinnPhong光照模型shader,用到了一盏平行光</br>
